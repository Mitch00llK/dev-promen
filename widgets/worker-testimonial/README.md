# Worker Testimonial Widget

This widget displays testimonials from workers with a background image, quote, and name.

## Directory Structure

- `class-worker-testimonial.php` - Main widget class file
- `includes/` - Directory containing control files
  - `controls-content.php` - Content tab controls
  - `controls-style.php` - Style tab controls
- `templates/` - Directory containing template files
  - `worker-testimonial-template.php` - Frontend template

## CSS

The CSS for this widget is located in:
`/plugins/promen-elementor-widgets/assets/css/widgets/worker-testimonial.css`

## Features

- Background image with overlay options
- Quote with customizable icon
- Worker name with split heading option
- Fully responsive design
- Multiple positioning and alignment options

## Usage

1. Add the widget to your Elementor page
2. Configure the content in the Content tab
3. Style the widget in the Style tab

## Customization

### Content Options

- Show/hide image
- Show/hide quote
- Show/hide name
- Split heading option for name

### Style Options

- Image height, fit, position, and overlay
- Quote icon color, size, and alignment
- Content position and alignment
- Typography for quote and name
